# Reconstruction-of-Hansen-s-High-Temperature-Air-Model
Attached is the Matlab code utilized in the paper "Reconstruction of Hansen’s High-Temperature Air Model" by A. Dunn, J. Ranstead, H. Ali, and S. Olcmen

Main script is Hansen_main.m which calls Hansen_Conductivity.m, Hansen_Transport.m, and Hansen_Viscosity.m
